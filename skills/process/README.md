# Process

Skills do workflow de desenvolvimento. Independentes de stack onde possível; com nuance Laravel onde faz diferença.

- **[grill-with-docs](./grill-with-docs/SKILL.md)** — entrevista que estressa o plano contra o domínio e atualiza CONTEXT.md/ADRs inline.
- **[to-prd](./to-prd/SKILL.md)** — sintetiza a conversa atual num PRD com template Laravel (Domain/Application/Presentation/Persistence) e publica como issue no GitHub.
- **[to-issues](./to-issues/SKILL.md)** — quebra um PRD em slices verticais Laravel (4 camadas) e abre uma issue por slice.
- **[tdd](./tdd/SKILL.md)** — red-green-refactor com Pest 3/4. Slice vertical Laravel (Migration → Action → Livewire → Pest) por ciclo.
- **[open-pr](./open-pr/SKILL.md)** — abre PR no GitHub: roda Pest+Pint, deriva título de Conventional Commits, monta corpo com issues fechadas e ADRs tocados.
